## 벡터의 정의

크기와 방향을 모두 가진 수량을 가르키는 말

## 벡터의 종류

* 힘 (방향 + 세기)
* 변위 (최종적인 이동 방향 + 거리)
* 속도 (방향 + 빠르기)
