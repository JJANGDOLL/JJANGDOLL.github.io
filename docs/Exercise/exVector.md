# 2. $u=(-1, 3, 2)$ 이고 $v = (3, -4, 1)$ 이라고 하자. 다음 계산을 수행하라.

## $u + v$

$u + v $

$ = (-1, 3, 2) + (3, -4 ,1)$

$ = (-1 + 3, 3 + -4, 2 + 1)$

$ = (2, -1, 3)$

&nbsp;<br/>

## $u - v$

$ u - v$

$ = (-1, 3, 2) - (3, -4, 1)$

$ = (-1 - 3, 3 - -4, 2 - 1)$

$ = (-4, 7, 1)$

&nbsp;<br/>

## $3u + 2v$

$ 3u + 2v$

$ = 3(-1, 3, 2) + 2(3, -4, 1)$

$ = (-3,9, 6) + (6, -8, 2)$

$ = (-3 + 6, 9 + -8, 6 + 2)$

$ = (-11, 1, 8)$

&nbsp; <br/>

## $-2u + v$

$ -2u + v$

$ = -2(-1, 3, 2) + (3, -4, 1)$

$ = (2, -6, -4) + (3, -4, 1)$

$ = (2 + 3, -6 + -4, -4 + 1)$

$ = (5, -10, -3)$
